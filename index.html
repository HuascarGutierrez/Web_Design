RULES IN HTML

  1. Not to design with HTML, just structure
  2. Not to use obsolete tags, because Google punishes those websites with old tags
  3. Make a semanticaly correct website
  4. Not to put styles on HTML ej. h1 style='color:red'
  5. For the same reason, not to use style tag
  6. Not to insert scripts in the HTML, just link it
